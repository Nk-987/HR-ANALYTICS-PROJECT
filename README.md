# HR Analytics Dashboard – Attrition Insights

## 📊 Project Overview
This project analyzes employee attrition data to identify key factors affecting employee retention using Python and Power BI.

---

## ⚙️ Tech Stack
- Python (Pandas)
- Power BI
- Excel

---

## 🔄 Workflow
Raw Data → Data Cleaning (Python) → Feature Engineering → Power BI Dashboard

---

## 🔍 Key Insights
- Employees with low income show highest attrition (~28%)
- Sales department has highest turnover (~20%)
- Higher salary significantly reduces attrition
- Job satisfaction directly impacts employee retention

---

## 🎯 Outcome
Built an interactive dashboard to support data-driven HR decisions and improve employee retention strategies.
